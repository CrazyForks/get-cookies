# 使用注意
整个代码打包下载回去，使用chrome扩展 加载已解压的扩展程序来使用
注意！使用前请自己打开background.js和manifest.json文件，

 "matches": [ "http://192.168.0.229/*","http://abc.myds.me/*"]
替换修改其中的ip字串符和域名字串为需要使用的签到平台的（任意即可）
